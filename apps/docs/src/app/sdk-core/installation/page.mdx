## Installation

Install ArkProject Core via your package manager or build from source.

## Package manager

Install the required packages.

```bash
pnpm add @arkproject/core starknet
```

## Requirements

ArkProject Core is optimized for modern browsers. It is compatible with the following browsers.

- Chrome 64+
- Edge 79+
- Firefox 67+
- Opera 51+
- Safari 12+
