# Tokens

Tokens are the heartbeat of the Ark NFT ecosystem, representing unique digital assets. {{ className: 'lead' }}

## The Token Model

The token model encapsulates the intricate details of individual tokens within a contract on the ArkProject platform. Each token is uniquely identified by its **token_id** and is tethered to a specific **contract_address**, which denotes the contract it belongs to. The **type** property further classifies the nature of the token within that contract.

Every token is associated with a **current_owner**, indicating the current holder of the token.

### Properties

<Properties>
  <Property name="token_chain_id" type="string">
    Unique id of the chain.
  </Property>
  <Property name="token_address" type="string">
    Unique identifier for the token contract.
  </Property>
  <Property name="token_id" type="string">
    Token ID.
  </Property>
  <Property name="current_owner" type="string">
    The address of the owner of the token.
  </Property>
  <Property name="listed_timestamp" type="i64">
    The listed timestamp of the token.
  </Property>
  <Property name="updated_timestamp" type="i64">
    Last event timestamp.
  </Property>
  <Property name="current_price" type="string">
    Price of the token.
  </Property>
  <Property name="quantity" type="string">
    Quantity of the token
  </Property>
  <Property name="start_amount" type="string">
    Price the owner wants to sell the token.
  </Property>
  <Property name="end_amount" type="string">
    Max price for a bid.
  </Property>
  <Property name="start_date" type="string">
    Start date of the sell.
  </Property>
  <Property name="end_date" type="string">
    End date of the sell.
  </Property>
  <Property name="broker_id" type="string">
    Broker where the token is on sale.
  </Property>
  <Property name="is_listed" type="boolean">
    True if the token is listed.
  </Property>
  <Property name="has_offer" type="boolean">
    True if the token has at least one offer.
  </Property>

</Properties>

## Retrieve a token {{ tag: 'GET', label: 'token/{contract_address}/{token_id}' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a token by providing its contract address and its token ID. Refer to the list at the top of this page to see which properties are included with token objects.

    ### Path parameters

    <Properties>
      <Property name="contract_address" type="string">
        Hexadecimal address of the contract.
      </Property>
      <Property name="token_id" type="string">
        Hexadecimal or decimal token ID.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/token/{contract_address}/{token_id}">

