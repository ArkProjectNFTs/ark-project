# Offers

Offers capture significant activities related to tokens, offering a detailed view of offers. {{ className: 'lead' }}

## The Offer Model

The offer model chronicles the various activities associated with tokens on the ArkProject platform. Each offer is tied to a specific **token_address** and **token_id**, pinpointing the exact token involved in the event.


### Properties
<Properties>
  <Property name="token_address" type="string">
    Unique identifier for the token contract.
  </Property>
  <Property name="token_id" type="string">
    Token ID.
  </Property>
  <Property name="current_owner" type="string">
    The address of the owner of the token.
  </Property>
  <Property name="current_price" type="string">
    Price of the token.
  </Property>
  <Property name="offers" type="array">
    Offers's array of the token.
  </Property>
  <Property name="offers.offer_maker" type="string">
      Who did the offer.
  </Property>
  <Property name="offers.offer_amount" type="string">
      Amount for the offer.
  </Property>
  <Property name="offers.offer_quantity" type="string">
      Number of token included in the offer.
  </Property>
  <Property name="offers.offer_timestamp" type="string">
      Timestamp of the offer.
  </Property>
</Properties>

## Retrieve offers for a specific token {{ tag: 'GET', label: '/token/{contract_address}/{token_id}/offers' }}
