# Ark Marketplace API

Welcome to the Ark Marketplace API, a robust platform designed to empower developers with tools to interact with the NFT ecosystem. {{ className: 'lead' }}

Our API provides a suite of endpoints tailored to cater to various needs, from fetching collection details to tracking portfolio. Below is a brief overview of our primary endpoints:

## Endpoints Overview

### Collections

- **List All Collections**: `/collections`
- **Retrieve a Specific Collection**: `/collections/{address}/{chain_id}`
- **List All Collections for a User's Portfolio**: `/portfolio/{user_address}/collections`

[Explore Collections Endpoints](/marketplace-api/collections)

### Tokens

- **List Tokens in a Specific Collection**: `/collections/{address}/{chain_id}/tokens`
- **List All Tokens for a User's Portfolio**: `/portfolio/{user_address}`

[Explore Tokens Endpoints](/marketplace-api/tokens)
