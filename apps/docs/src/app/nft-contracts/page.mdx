# NFT Contracts

Explore NFT collections within the Starknet ecosystem effortlessly.

## Contract Model

Our NFT contract framework is designed to simplify your interaction with contracts in the ArkProject ecosystem. Each contract is identified by a unique address, serving as its digital signature. The type of the contract provides insight into its function or role.

Key details like the name and symbol make each contract easily identifiable, reflecting its essence or theme. The optional image attribute helps visualize the contract, making it more recognizable.

### Properties

<Properties>
  <Property name="contract_address" type="string">
    Unique identifier for the contract.
  </Property>
  <Property name="contract_type" type="string">
    The type of contract.
  </Property>
  <Property name="name" type="string">
    The name of the contract.
  </Property>
  <Property name="symbol" type="string">
    The symbol for the contract.
  </Property>
  <Property name="image" type="string">
    The image representing the contract. It can be nullable.
  </Property>
</Properties>

## List all NFT contracts {{ tag: 'GET', label: '/v1/contracts' }}

<Row>
  <Col>

    This endpoint allows you to retrieve all Starknet NFT contracts.

    ### Query string parameters

    <Properties>
      <Property name="cursor" type="string">
        The cursor to get the next page. Retrieved from previous request that may be paginated.
      </Property>
      <Property name="limit" type="number">
        The limit on the number of items returned per request, defaulting to 100, with a 1MB data cap. (Refer to the [pagination](/pagination) section below for more details.)
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/contracts">

    ```bash {{ title: 'cURL' }}
    curl https://api.arkproject.dev/v1/contracts \
      -H "x-api-key: <INSERT_API_KEY_HERE>"
    ```

## Retrieve an NFT Contract {{ tag: 'GET', label: '/v1/contracts/{contract_address}' }}

<Row>
  <Col>

    This endpoint allows you to retrieve an nft contract by providing its contract address. Refer to [the list](#the-collection-model) at the top of this page to see which properties are included with collection objects.

    ### Path parameters

    <Properties>
      <Property name="contract_address" type="string">
        Hexadecimal address of the contract.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/contracts/{contract_address}">

    ```bash {{ title: 'cURL' }}
    curl https://api.arkproject.dev/v1/contracts/{contract_address} \
      -H "x-api-key: <INSERT_API_KEY_HERE>"
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "cursor": null,
      "result": {
          "contract_address": "0x1111",
          "symbol": "EVE",
          "name": "Everai",
          "image": null
      }
    }
    ```

  </Col>
</Row>
