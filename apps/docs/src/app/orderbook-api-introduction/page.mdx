# Ark Orderbook API

Welcome to the Ark Orderbook API, a robust platform designed to empower developers with tools to interact with the NFT ecosystem. {{ className: 'lead' }}

Our API provides a suite of endpoints tailored to cater to various needs, from fetching token details to tracking events. Below is a brief overview of our primary endpoints:

## Endpoints Overview

### Tokens

- **Retrieve a Specific Token**: `/token/{contract_address}/{token_id}`

[Explore Tokens Endpoints](/orderbook-api/tokens)

### Collections

- **List Tokens in a Collection**: `/tokens/collection/{collection_id}`

[Explore Tokens Endpoints](/orderbook-api/tokens)

### Events

- **Retrieve Events for a Specific Token**: `/token/{contract_address}/{token_id}/history`

[Explore Events Endpoints](/orderbook-api/events)

### Offers

- **Retrieve Offers for a Specific Token**: `/token/{contract_address}/{token_id}/offers`

[Explore Offers Endpoints](/orderbook-api/offers)

### Owners

- **List All Tokens for an account**: `/tokens/{account_address}`

[Explore Tokens Endpoints](/orderbook-api/tokens)
