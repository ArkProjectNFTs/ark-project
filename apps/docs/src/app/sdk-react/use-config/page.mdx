# useConfig

Hook for getting [Config](/sdk-core/configuration) from nearest [ArkProvider](/sdk-react/getting-started#react-next-js-application-setup-with-stark-net-provider).

## Import

```typescript
import { useConfig } from '@ark-project/react'
```

## Usage

```typescript
import React from 'react'
import { useConfig } from '@ark-project/react'

function App = () => {
  const config = useConfig()

  return (
    <div>
      {config && <pre>{JSON.stringify(config, null, 2)}</pre>}
    </div>
  )
}
```

## Return Type

`Config | undefined`

[Config](/sdk-core/configuration) or `undefined` if no provider found.
